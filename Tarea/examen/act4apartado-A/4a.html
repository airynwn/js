Para crear un proyecto en React hay que tener instalado previamente NodeJS y NPM.

Esto se comprueba escribiendo los comandos: npm -v y node -v

  1. Situarse en la carpeta en la que se quiere crear el proyecto.
  2. Escribir en la terminal: npm create vite@latest nombredelproyecto
  3. Elegir 'React' y 'Javascript' con las flechas y la tecla Enter.
  4. Una vez creado, hay que situarse en la carpeta: cd nombredelproyecto (tendrá el nombre que le pusimos en el comando).
  5. Ejecutar el siguiente comando para cargar lo necesario para el funcionamiento del proyecto: npm install
  6. Crear los componentes que se quieran mostrar en el proyecto e incorporarlos desde App.jsx.
  7. Por último, ejecutar el siguiente comando para empezar a ver el proyecto en localhost: npm run dev
